# Changelog

## v6.1.4
- Renamed the first-page Staff Generator tab to Staff Preview.
- Made Email Address read-only on the staff page.
- Email Address now auto-generates immediately from Email Username plus the configured district domain.
- Updating the district domain in Admin now refreshes the staff email preview automatically.

## Version 6.1.3
- Moved Extension from Personal Information to Building & Contact.
- Renamed Desk Phone to Phone Number.
- Moved Include QR Code from Message, Logo & Options to Website & Scheduling because QR codes are tied to optional links.
- Updated project metadata and staff-form documentation to v6.1.3.

## Version 6.1.2
- Merged the newer v5.5 accessibility/branding improvements into the v6.1.1 package.
- Added WCAG/ADA color contrast checking.
- Added optional automatic brand-color adjustment for WCAG/ADA readability.
- Added live contrast feedback in the Branding tab.
- Improved accessibility markup for tabs, panels, labels, and selection cards.
- Added clearer hosted-logo warning behavior for PNG export compatibility.
- Updated project metadata to K12 Signature Building v6.1.2.

## Version 6.1.1
- Fixed the Executive signature style so it is visually distinct from Standard.
- Executive now uses a framed card layout with a left brand bar, uppercase name treatment, and stronger district footer rule.

## Version 6.1
- Fixed logo upload embedding.
- Uploaded logos are stored as Base64 for portable staff exports and PNG rendering.
- Added shareable package documentation.

## Version 6.0
- Packaged as K12 Signature Building.
- Added configurable district builder and staff generator export.
